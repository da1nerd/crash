A entity-component-system framework loosly ported from [Ash](https://github.com/richardlord/Ash).

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     crash:
       github: neutrinog/crash
   ```

2. Run `shards install`
